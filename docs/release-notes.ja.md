# MoMorph — リリースノート

MoMorph（Plugin・Web・MCP Server）の最新アップデート。

## 2026-05-21

**✨ 新機能**

- **[Pro アカウントのみ] Figma URL の初回入力のみで自動記憶** — いつでも Settings からリセット可能。
- **Screen ID で検索** — Screen List と Screen Set の検索ボックスで、画面名に加えて Screen ID による部分一致検索（大文字小文字を区別しない）に対応。
- **アーカイブアイテムを非表示（Web）** — View All Spec 画面でアーカイブ済みアイテムを自動的に非表示にし、active アイテムのみを表示。

**🔧 改善**

- **画面詳細の表示を改善** — タイトルのフォントサイズを縮小し、コンテナ幅に応じて伸縮、必要な場合のみ省略表示。Edit タグボタンを最後のタグの直後にインライン配置（タグが少ない場合の余白を解消）。
- **AI 仕様生成のキャンセル機能** — 生成中のキャンセルボタンをアイテム単位・バッチ単位で追加（全キャンセル or キュー待ちのみ）。バッチ完了後に成功/失敗件数のサマリートーストを表示。
- **フレーム選択時のナビゲーション** — Figma 上でフレームを選択すると、Screen List で自動スクロール＆ステータスタブの自動切替がよりスムーズに。
- **アイテムとプレビューの双方向ハイライト** — リストのアイテムをクリックするとプレビュー上のラベルがハイライト、その逆も対応。

**🐛 修正**

- **Copy Specs** — アイテムの順序、アーカイブ状態、UI Part の表示を正しく保持。コピー後の UI フリッカリングを解消。
- **削除済みフレームを含む Screen Set** — Figma 上でリンク済みフレームを削除した際に発生していたラグ、不整合なステータス、遅い読み込み、誤った自動リンク、Screen の消失を修正。
- **AI テストケース生成** — ステータス変更が未保存の状態で Gen TC ボタンが有効になりタイムアウトする問題を修正。デザインがない画面でも安定動作。
- **GitHub 連携** — 管理者権限がないリポジトリ接続時のエラーメッセージをより分かりやすく改善。GitHub Issue 翻訳作成時の不要なフレーム自動生成を解消。
- **Screen Spec** — ラベル付き新規フレームでの未保存変更モーダル誤表示を修正。Item Detail へ遷移して戻った際のレイヤーリンク消失を修正。重複ラベル/UI Part の誤検出によるアイテム誤削除を修正。
- **プレビューとバッジ** — 翻訳後の画像ズーム問題、Text タイプの UI Part にバッジが表示されない問題、Plugin リフレッシュ後に UI Part 列が空白になる問題を修正。
- **Screen List** — GitHub アップロード後にタブハイライトが Design & Specs に固定される問題を修正。Overview がある場合のフレームステータス表示を修正。
- **Item Spec（Web）** — Screen Set フローでのページネーション誤表示と保存エラーを修正。Figma で新規追加したフレームが Web に同期されない問題を修正。
