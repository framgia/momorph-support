# MoMorph — リリースノート

MoMorph（Plugin・Web・MCP Server）の最新アップデート。

## 2026-06-19

**🔧 改善**

- **画面仕様の入力をより柔軟に** — `No`・`Item Name`・`UI Part` の **3 つのうちいずれか 1 つ** を入力すれば、アイテムの保存・ステータス変更が可能に。**Item List** と **Input Spec** の両画面で、Plugin・Web 共通で適用。
