# MoMorph — リリースノート

MoMorph（Plugin・Web・MCP Server）の最新アップデート。

## 2026-06-25

**✨ 新機能・改善**

- **メンテナンス後の更新通知** — メンテナンス終了後、最新版を取得するためのリロード（Plugin は Figma ファイル、Web はページ）を促す通知を表示。閉じてそのまま作業を継続できます。
- **大量アイテムの AI 仕様生成を安定化** — 多数のアイテムを一括生成しても安定（大量バッチでの 502/503 エラーを解消）。生成に少し時間がかかる旨の案内を表示。
- **大規模ファイルのフレーム同期を強化** — 大規模 Figma ファイル（200+ フレーム）の同期を堅牢化。バッチ処理とリトライにより、タイムアウトや通信断によるデータ損失を防止。

**🐛 修正**

- **レイヤー階層に応じた自動採番** — Figma のレイヤー順（上から下）に従って採番されるように修正。ドラッグでの並べ替えも安定。
- **Screen Detail プレビュー（Web）** — 画面を開いた際にプレビュー画像がちらつく不具合を修正。
- **戻る操作（Plugin）** — フレームセット一覧へ戻る際の白画面を修正。
- **Destination フィールド（仕様フォーム）** — ドロップダウンに存在しないフレームが表示されなくなり、必須の Destination を空欄のまま保存できないように修正。
- **同期後のアイテムステータス** — AI 生成アイテムが MoMorph への同期後に正しく **Done** になるよう修正。
- **シート上の Item Type** — AI 生成のボタンアイテムが、シート同期時に `Button-Icon/Text` ではなく `Button` と表示されるよう修正。
- **辞書ファイルのアップロード** — 辞書 `.xlsx` ファイルのアップロード時のエラーを修正。
- **Issue アップロード** — GitHub への issue アップロード後にアイテムの `No` と `名称` が消える不具合を修正。
- **メディアアップロード** — メディアアップロードが失敗する不具合を修正。
