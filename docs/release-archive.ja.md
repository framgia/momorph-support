## 2026-04-23

**✨ 新機能**

- **常に編集可能な画面仕様** — 表示モードと編集モードの分離を廃止しました。テーブルはデフォルトで常に編集モードになり、項目番号は自動インクリメントではなく手動で入力するようになりました。
- **仕様を CSV 形式でダウンロード** — 画面仕様データを CSV 形式でエクスポートおよびダウンロードできるようになりました。

**🔧 改善点**

- **Figma フレームの一括削除/取り消し** — プラグインが Figma 上で複数のフレームを一度に削除および取り消す機能をサポートするようになりました。
- **デザイン項目の順序** — 画面仕様上の項目は、Figma のデフォルトのレイヤーツリー順序に従うようになりました。
- **AI による項目生成** — より正確な AI 生成仕様のために、`node_link_id` リストがペイロードに含まれるようになりました。
- **検索キーワードの保持** — 画面リストの検索キーワードが、ナビゲーション中の URL クエリパラメータに保持されるようになりました。
- **GA4 アナリティクス** — Figma ファイルごとのフィルタリングのために、すべての GA4 イベントに `file_key` を追加しました。

**🐛 バグ修正**

- **データ同期とアップロード：** Google スプレッドシートから MoMorph への同期エラーを修正、アーカイブされたアイテムが GitHub へのアップロードに含まれなくなり、「デザインなし」の画面がプラグインアップロード後に正しく「開発中」に移動するようになりました。Web 上で重複した画面が作成されなくなりました。
- **削除/アーカイブされたアイテム：** プラグインの再読み込み後および Google スプレッドシートへのアップロード後に削除されたアイテムが再び表示される問題を修正しました。プレビューの再読み込みアイコンをクリックした後、アーカイブされたアイテムが復元できない問題を修正しました。
- **画面仕様の編集：** Figma レイヤーを選択した際に新しい行の入力がリセットされ、自動保存されない問題を修正しました。フレームセットで仕様を切り替えた際にアイテムリストがキャッシュされる問題を修正しました。画面仕様の再読み込み時に 404 エラーが発生する問題を修正しました。
- **プレビューとデザインのリンク：** リンクされていない UI パーツを含むフレームのプレビューが正しく表示されない問題を修正しました。画面セットから初めて画面詳細にアクセスした際にプレビューが同期されない問題を修正しました。リンクされていないデザインが、手動リンク保留状態のままではなく、自動生成されたボックスに自動的に再割り当てされる問題を修正しました。
- **Google スプレッドシートへのアップロード：** アイテムタイプのフィルタリングの問題（Web ではボタンのみが表示され、プラグインでは他のアイテムが表示される）を修正しました。画像のズームの不一致によりエッジラベルが隠れてしまう問題を修正しました。
- **プラグインの接続性：** 削除/アーカイブを妨げる断続的なフェッチの競合を修正しました。GitHub 再接続後にリポジトリが編集できない問題を修正しました。Figma ノードを選択した際にキャンセル/エスケープキーが表示されない問題を修正しました。
- **その他：** 画面詳細の読み込みに失敗する問題、画面セットリストが完全にスクロールできない問題、トークンの有効期限が切れた際に Google 接続が切断される問題、新しい mms ノードプレフィックスを適用した後のメディアスキャン、AI 再翻訳中の辞書アップロードのタイムアウトを修正しました。また、VSCode 拡張機能のデータアクセス権限に関するセキュリティ問題を修正しました。

## 2026-04-09

**✨ 新機能**

- **プレビューの仕様ラベル表示切り替え** — デザインプレビューエリアで仕様ラベルの表示・非表示をワンクリックで切り替えられるようになりました。デザインをすっきり確認したい場合にご活用ください。

**🔧 改善・変更**

- **AI 仕様・テストケース生成の精度向上** — AI 生成時に画面の概要情報（screen overview）がコンテキストとして送信されるようになり、生成結果の精度が向上しました。
- **Design ID クリックで自動スクロール** — プレビュー上の Design ID をクリックすると、リスト内の対応アイテムに自動スクロールしてハイライト表示されます。
- **Screen ID コピー時のフィードバック表示** — Screen ID をコピーした際に、操作完了を知らせるトースト通知が表示されるようになりました。
- **ツールチップの動作改善** — 全ツールチップに 400ms のディレイとフェードインエフェクトを追加し、より洗練された操作感になりました。
- **レイヤー ID 検出の改善** — Plugin の初回読み込み時に `mms_id_` プレフィックスのレイヤーが正しく認識されるようになりました。
- **URL 統一** — 画面 URL が全体を通じて `frame_id` から `screen_id` に統一されました。

**🐛 バグ修正**

- プレビュー表示、仕様同期、アーカイブフレーム、UI Parts、Input Spec 保存、AI 生成ステータス、GitHub リポジトリ検索など、いくつかの不具合を修正しました。

## 2026-03-20

**🆕 新機能**

- 画面一覧が 4 つのステータスタブ（設計中 / 開発中 / レビュー中 / 完了）に分かれ、件数バッジで一目確認できます。
- 不要な画面を「アーカイブ」して管理できます。旧バージョンで非表示だった画面は自動的に移行されます。
- デザイン未完成（No UI / In Design）の画面も詳細をクリックで閲覧可能になりました。
- 項目仕様の説明欄で Markdown が使えます。ツールバーから太字・リスト・コードブロックなどを簡単入力できます。Plugin・Web 間で完全同期。
- 項目仕様フォームで ID 入力と Figma レイヤーへの直接リンク設定が可能になりました。リンク時はレイヤー名に「mms_」が自動付与されます。
- Figma レイヤーリンク時のプレフィックスが「mms_」に統一されました。旧形式は自動変換されます。

**✨ 改善**

- 画面セットの 4 画面（一覧・追加・詳細・編集）の UI を全面リニューアル。日本語・英語・ベトナム語に対応。
- 画面仕様のプレビューエリアに全画面表示モードと仕様ナンバリング設定ボタンを追加しました。
- 画面詳細から設計・仕様・開発のステータスを更新でき、変更は画面一覧にリアルタイム反映されます。

**🔧 バグ修正**

- 仕様アップロード時に Google アカウントが期限切れの場合、明確なエラーと再接続ボタンを表示するよう改善しました。
- AI 仕様生成後、ページリロードなしで内容がすぐに UI に反映されるようになりました。
- タグ管理画面で保存後に「ページを離れますか？」が誤表示される問題を修正しました。
- AI 生成失敗時にエラートーストが表示され、ステータスが正しく更新されるようになりました（generating → error / timeout）。
- タグ管理画面のボタンデザインをアプリ全体と統一しました。

**🗑️ 廃止**

- 設定画面から「デフォルトモード」の選択肢を削除しました（ステータスタブ表示に移行）。
- 画面仕様作成中のフレームロック機能を廃止しました。Figma 上のデザインを自由に編集できます。
- 仕様未作成の項目への AI 生成時、確認モーダルを廃止しました（直接設定画面が開きます）。
